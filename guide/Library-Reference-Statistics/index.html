    # 库引用统计

    # 使用方式

    主页的第二个 Tab 即为库引用统计，在此页面，您可以方便地看到设备中所有第三方库的引用情况，它是按照引用次数排序的。如下图所示。

    由于性能考虑，列表默认加载原生库类型，您可以点击右上角的更多按钮切换展示类型。

    如果您觉得列表项目太多，且库引用数量较少的库无需查看时，您可以在设置项中设置 库引用阈值。

    滑动数量条来改变阈值。
